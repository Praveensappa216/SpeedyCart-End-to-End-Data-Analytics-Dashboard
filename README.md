# SpeedyCart-End-to-End-Data-Analytics-Dashboard
End-to-end data analytics project using Python, SQL (SQLite), and Power BI to analyze orders, customers, city performance, and delivery efficiency for a quick-commerce platform. 

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- SQL (SQLite)
- Power BI Desktop
- VS Code
